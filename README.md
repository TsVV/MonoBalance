# BankBalance

Додаток для перегляду балансу карток Monobank з конвертацією валют.
Реалізовано на Electron з використанням системного сховища паролів через **keytar** для безпечного збереження токена.

## Можливості

* Перегляд балансу вибраної картки.
* Автоматична конвертація у вибрану валюту.
* Збереження історії змін балансу.
* Безпечне збереження токена у Windows Credential Manager / macOS Keychain / Linux Secret Service.
* Підтримка кількох валют і вибору картки.

## Вимоги

* Node.js 20 або новіше.
* npm (постачається з Node.js).
* Windows 10/11 (для інших ОС можливе збирання вручну).

## Запуск у режимі розробки

```bash
npm install
npm start
```

## Збірка інсталятора (Windows)

```bash
npm run dist
```

Інсталятор буде створено в теці `dist/`.

> Якщо під час збирання зʼявиться помилка з правами на створення символічних посилань,
> запусти термінал від імені адміністратора або увімкни **Developer Mode** у Windows.

## Налаштування

1. У розділі **Налаштування** введи токен Monobank і натисни **«Завантажити картки»**.
2. Після збереження токена поле стане неактивним і показуватиме `•••• збережено`.
3. Обери картку, валюту та натисни **«Оновити баланс»**.

## Безпека

* Токен Monobank **не зберігається у файлах чи localStorage**.
* Усі запити до API виконуються у головному процесі Electron.
* Дані токена шифруються операційною системою.
